# /entity 커맨드

Entity + Repository + Service + Controller 스캐폴딩을 생성한다.

## 실행 순서
1. entity 패키지에 {Name}.java 생성 (BaseEntity 상속)
2. repository 패키지에 {Name}Repository.java 생성
3. service 패키지에 {Name}Service.java 생성
4. controller 패키지에 {Name}Controller.java 생성
5. dto 패키지에 {Name}RequestDto.java / {Name}ResponseDto.java 생성
